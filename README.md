# 🌿 MapBiomas Bolivia – Tools & Resources

Explore the specialized resources and tools developed for analyzing and monitoring land use and land cover (LULC) in Bolivia, based on the official MapBiomas methodology and integrating advanced geospatial analysis approaches.

This repository serves as the entry point to the Earth Engine scripts and assets developed for the [**MapBiomas Bolivia**](https://bolivia.mapbiomas.org/) collection. Below you will find links to the main code repositories used in our workflows.

---

## 📂 Available Repositories

| Repository | Description |
|------------|-------------|
| [**bolivia-coverage**](https://github.com/mapbiomas/bolivia-coverage) | 🗺️ Land use and land cover classification scripts for Bolivia. Implements the official MapBiomas mapping methodology using Google Earth Engine. |
| [**water-monitoring**](https://github.com/mapbiomas/bolivia-water) | 💧 Water monitoring toolkit – adapted for Bolivia to analyze surface water dynamics, floods, and permanent water bodies. |

---

## 🚀 Getting Started

Each repository contains its own detailed documentation. To run the scripts you will need:

- A Google Earth Engine account.
- Access to the required assets (some are public, others may need to be copied to your own account).
- Follow the instructions inside each repository’s `README.md` to set up the necessary modules and output paths.

If you are new to MapBiomas workflows, please refer to the [official MapBiomas documentation](https://mapbiomas.org/en).
